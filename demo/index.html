What is kyro?

Post-naive RAG, in one library.HyDE · ColBERT · CRAG · Self-RAG · GraphRAG · ReAct · semantic cache · SSE streaming · multi-tenancy.

Kyro is a production-grade Retrieval-Augmented Generation pipeline with the post-naive primitives the literature actually demands — corrective and self-reflective retrieval, multi-step decomposition, agentic ReAct, and lightweight GraphRAG — fronted by an SSE-streaming agent endpoint and a distributed semantic cache that collapses 20–40% of production query traffic onto Redis. Multi-tenant by design, observable via OpenTelemetry, deployable via Helm. Built so the next engineer inherits a vessel — not a graveyard.
Capability matrix

Five primitives, one runtime.

⚡

RAG Agent

Bounded ReAct loop: thought → action → observation. Tool-traced. Timeout-enforced. Returns 504 deterministically on overrun.

🧠

Semantic Cache

Cosine-gated lookup with LRU eviction. ~3 ms p50 cache hit vs 800 ms LLM cold path. Threshold tuned per workload.

🔀

Redis Backend

Tenant-namespaced HASH + LRU ZSET. Survives pod restarts. Optional TTL. K3 graceful fallback to in-memory.

📡

SSE Streaming

/agent/query/stream: one frame per ReAct step, then a result frame, then a telemetry frame, then [DONE].

🐍

Typed SDK

Frozen-dataclass response models, structured exception hierarchy, context-managed lifecycle. Pure httpx — no rag-framework lock-in.

🔐

Multi-tenant + Auth

JWT-derived tenant_id auto-scopes Qdrant filters and cache keys. API-key or Bearer. Sliding-window rate limit. All optional.

blocking agent_query 
from konjoai.sdk import KonjoClient

with KonjoClient("https://kyro.acme.io", api_key="sk-...") as client:
    response = client.agent_query("What is the refund policy?", top_k=3)
    print(response.answer)
    print([s.action for s in response.steps])
streaming agent — print steps as they arrive 
for ev in client.agent_query_stream("How do refunds work?"):
    if ev.type == "step":
        print(f"  → " + ev.data["action"] + "  " + ev.data["observation"][:80])
    elif ev.type == "result":
        print("\n" + ev.data["answer"])
